# hy.common.modbus



---
#### 特别说明
本类库中的com.serotonin包中的所有类均来自于 https://github.com/MangoAutomation/modbus4j 。

并没有对它做任何的更改操作。只因为它没有发布到Maven中央，无法引用它。

只为了更好的使用，才将它的源码包含在本类库中的。

没有冒犯原作者的意思，仅仅是了更方便的使用。
